# Tech-Line Industries: Quality & Training Analytics 🚀

### مشروع تحليل بيانات: أثر الاحتياجات التدريبية ورأس المال النفسي على الجودة الشاملة
This project integrates **Python Data Analytics** with **Human Resource Management** to analyze the impact of Training and Psychological Capital on TQM in a tech-industrial environment.

## 📌 Project Overview
- **Case Study:** Tech-Line Industries.
- **Goal:** Identifying training gaps and measuring the mediating role of Psychological Capital (PsyCap).
- **Tools:** Python (Pandas, Statsmodels, Seaborn, Matplotlib).

## 📊 Methodology (Google Data Analytics Framework)
1. **Ask:** How can we ensure TQM through strategic training?
2. **Prepare & Process:** Synthetic data generation for 500 employees simulating real-world tech departments.
3. **Analyze:** Multiple Linear Regression & Correlation Matrix.
4. **Share:** Visualizing findings for strategic decision-making.

## 📈 Key Findings
- **R-squared (0.67):** The model explains 67% of the variance in Quality performance.
- **P-Value (0.000):** Both training and PsyCap have a statistically significant impact on TQM.
